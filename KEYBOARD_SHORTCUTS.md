# Keyboard Shortcuts

This document lists all available keyboard shortcuts for the Celestial Vehicle Propulsion Simulator.

## Time Controls

| Key | Action |
|-----|--------|
| `Space` | Pause / Play simulation |
| `1` | Set time speed to 1x |
| `2` | Set time speed to 100x |
| `3` | Set time speed to 10,000x |
| `4` | Set time speed to 100,000x |
| `5` | Set time speed to 500,000x |

## Camera Controls

| Key | Action |
|-----|--------|
| `C` | Cycle through camera modes (Free → Follow Spaceship → Planet Focus) |
| Mouse Drag | Rotate camera view |
| Mouse Wheel | Zoom in / out |

## Display Options

| Key | Action |
|-----|--------|
| `O` | Toggle orbital paths visibility |
| `L` | Toggle planet/asteroid labels |
| `F` | Toggle FPS monitor |

## Navigation

| Key | Action |
|-----|--------|
| `Esc` | Close panels / Cancel actions |

## Notes

- Keyboard shortcuts are disabled when typing in input fields
- Shortcuts are disabled during propulsion selection
- All shortcuts are case-insensitive
